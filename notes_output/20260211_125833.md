# Boundary of a Set in $(M,d)$

**Definition:** For a given set $A \subset M$, the boundary of $A$ is
$$\partial A = \text{cl}(A) \cap \text{cl}(A^c)$$

**Examples:**
$$\partial M = \emptyset, \quad \partial \mathbb{Z} = \emptyset, \quad \partial \mathbb{Q} = \mathbb{R}$$

**Lemma 1:** $x \in \partial A \iff \forall \varepsilon > 0, B_{\varepsilon}(x) \cap A \neq \emptyset \text{ and } B_{\varepsilon}(x) \cap A^c \neq \emptyset$.

**Proof (for $\Rightarrow$):**
$$x \in \partial A \iff x \in \text{cl}(A) \land x \in \text{cl}(A^c)$$

**Lemma 2:** $\partial A = \text{cl}(A) \setminus \text{int}(A) = \overline{A} \cap (\text{int}(A))^c$.

**Proof (showing $(\text{int}(A))^c = \overline{A^c}$):**
$$(\text{int}(A))^c = \overline{A^c} \text{ because }$$
$$x \in \text{int}(A)^c \iff \forall \varepsilon > 0: B_{\varepsilon}(x) \cap A^c \neq \emptyset$$
$$\iff x \in \overline{A^c}.$$
