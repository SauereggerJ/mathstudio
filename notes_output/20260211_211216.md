## 8) Eigenvalues of Idempotent Operators

Suppose $P \in \mathcal{L}(V)$ is such that $P^2=P$. Prove that if $\lambda$ is an eigenvalue of $P$, then $\lambda=0$ or $\lambda=1$.

**Solution:**
Let $v \in V$ be an eigenvector corresponding to $\lambda$, so $Pv = \lambda v$ and $v \neq 0$. 

$$P(Pv) = P(\lambda v) = \lambda (Pv) = \lambda (\lambda v) = \lambda^2 v$$
Since $P^2=P$, we have $P^2v = Pv = \lambda v$. 

Thus, we must have $\lambda^2 v = \lambda v$, which implies $(\lambda^2 - \lambda)v = 0$. Since $v \neq 0$, we must have $\lambda^2 - \lambda = 0$, so $\lambda(\lambda - 1) = 0$. Therefore, $\lambda = 0$ or $\lambda = 1$.

## 9) Eigenvalues of the Derivative Operator on Polynomials

Define $T: \mathcal{P}(\mathbb{R}) \to \mathcal{P}(\mathbb{R})$ by $T_P = P'$. Find all eigenvalues and eigenvectors of $T$.

**Solution:**
We seek $\lambda$ and a non-zero polynomial $P$ such that $TP = \lambda P$, meaning $P' = \lambda P$.

If $P$ is a non-zero constant polynomial, $P(x) = c \neq 0$, then $P'(x) = 0$. The equation becomes $0 = \lambda c$. Since $c \neq 0$, we must have $\lambda = 0$.

If $\lambda \neq 0$, let $n = \deg(P) \ge 0$. Then $\deg(P') = n-1$ (if $n \ge 1$), while $\deg(\lambda P) = n$. Since the degrees must match, this is only possible if $P'=0$, forcing $P$ to be constant, leading back to $\lambda=0$. 

Therefore, the only eigenvalue is $\lambda = 0$. The eigenvectors are the constant polynomials, $E_0 = \mathrm{span}\{1\}$.

## 10) Eigenvalues of $x P'(x)$ on $P_4(\mathbb{R})$

Define $T \in \mathcal{L}(P_4(\mathbb{R}))$ by $(T P)(x) = x P'(x)$ for all $x \in \mathbb{R}$. Find all eigenvalues and eigenvectors.

**Solution:**
We test the basis elements $P(x) = x^k$ for $k \in \{0, 1, 2, 3, 4\}$. (We define $x^0 := 1$).

$$(TP)(x) = x P'(x) = x \cdot (k x^{k-1}) = k x^k = k P(x)$$
Thus, for $P_k(x) = x^k$, we have $T P_k = k P_k$. 

The eigenvalues are $\lambda = k$ for $k \in \{0, 1, 2, 3, 4\}$. The corresponding eigenvectors are $P_k(x) = x^k$.
Since $\dim(P_4(\mathbb{R})) = 5$ and we found $5$ distinct eigenvalues with corresponding linearly independent eigenvectors, these are all of them.

## 11) Proximity of Non-Eigenvalues to Eigenvalues

Suppose $V$ is finite dimensional. $T \in \mathcal{L}(V)$, and $\alpha \in \mathbb{F}$. Prove that there exists $\delta > 0$ such that $T - \lambda I$ is invertible for all $\lambda \in \mathbb{F}$ such that $0 < |\alpha - \lambda| < \delta$.

**Solution:**
Since $V$ is finite dimensional, $T - \lambda I$ is invertible if and only if $\lambda$ is not an eigenvalue of $T$. Since $V$ is finite dimensional, the set of eigenvalues $\sigma(T)$ is finite. 

Let $\sigma(T) = \{\lambda_1, \lambda_2, \dots, \lambda_m\}$. We analyze $\alpha$ relative to $\sigma(T)$.

**Case 1: $\alpha$ is not an eigenvalue, $\alpha \notin \sigma(T)$.**

Define $\Delta := \min \{|\alpha - \lambda_i| : \lambda_i \in \sigma(T)\}$. Since $\sigma(T)$ is finite and $\alpha \neq \lambda_i$ for all $i$, $\Delta > 0$.
If we choose $\delta = \Delta$, then for any $\lambda$ such that $0 < |\alpha - \lambda| < \delta$, it implies that $|\alpha - \lambda| < |\alpha - \lambda_i|$ for all $i$. This means $\lambda$ cannot be any of the eigenvalues $\lambda_i$. Thus $T - \lambda I$ is invertible.

**Case 2: $\alpha$ is an eigenvalue, $\alpha \in \sigma(T)$.**

Let $\sigma'(T) = \sigma(T) \setminus \{\alpha\}$ be the set of eigenvalues of $T$ distinct from $\alpha$. 

If $\sigma'(T)$ is empty, $\sigma(T) = \{\alpha\}$. We can choose any $\delta > 0$ (e.g., $\delta=1$). If $0 < |\alpha - \lambda| < \delta$, then $\lambda \neq \alpha$, so $\lambda \notin \sigma(T)$, and $T - \lambda I$ is invertible.

If $\sigma'(T)$ is non-empty, define $\Delta_\alpha := \min \{|\alpha - \lambda| : \lambda \in \sigma'(T)\}$. Since $\sigma'(T)$ is finite and $\alpha \neq \lambda$ for all $\lambda \in \sigma'(T)$, $\Delta_\alpha > 0$.

Choose $\delta = \Delta_\alpha$. If $0 < |\alpha - \lambda| < \delta$, then $|\alpha - \lambda| < \Delta_\alpha$. This implies $\lambda$ is not any eigenvalue in $\sigma'(T)$. Since $\lambda$ cannot equal $\alpha$ (because $|\alpha - \lambda| > 0$), $\lambda$ is not an eigenvalue of $T$. Therefore, $T - \lambda I$ is invertible.

In all cases, we find a suitable $\delta > 0$ such that the open ball around $\alpha$ of radius $\delta$ contains no eigenvalues, except possibly $\alpha$ itself (which is excluded by the condition $0 < |\alpha - \lambda|$). This yields the statement.
