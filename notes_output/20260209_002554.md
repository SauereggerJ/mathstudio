Every $v \in \mathbb{K}^n$ defines a $v'(w) := v_1w_1 + \dots + v_nw_n$, with $w_1, \dots, w_n$ basis of $V$.

Example: $V = P_{n-1}(\mathbb{R})$ and $v = (v_1, \dots, v_n) \in \mathbb{K}$.

If $V$ is a finite dimensional vector space and $B = v_1, \dots, v_n$ is a basis of $V$, then the linear map $\varphi_B(v) = (x_1, \dots, x_n)$ with $v = x_1v_1 + \dots + x_nv_n$ is an isomorphism from $V$ to $\mathbb{K}^n$.
This isomorphism of course depends on $B$.
If $B' = v'_1, \dots, v'_n$ is another basis of $V$ with $\varphi_{B'}$ as its coordinate isomorphism, we get

$$\psi_{B'B} = \varphi_{B'}^{-1}(\varphi_B(v)) \text{ is a function from } V \to V.$$

1. Is this function linear?

*   $\psi_{B'B}(v_1 + v_2) = \varphi_{B'}^{-1}(\varphi_B(v_1 + v_2)) = \varphi_{B'}^{-1}(\varphi_B(v_1) + \varphi_B(v_2)) = \psi_{B'B}(v_1) + \psi_{B'B}(v_2)$
*   $\alpha \cdot \psi_{B'B}(v_1) = \dots = \psi_{B'B}(v_1 \alpha)$.

$\implies \checkmark$

2. What is the null space of the operator $\psi_{B'B}$?

$$\psi_{B'B}(v) = 0 \quad \varphi_{B'}^{-1}(v') = 0 \iff v' = 0 \text{ because } \varphi_{B'}^{-1} \text{ is an isomorphism}.$$
$$\varphi_B(v) = 0 \iff v = 0.$$

Therefore, $\text{null } \psi = \{0\}$.

$$\text{Therefore, } \psi_{B'B} \text{ is an isomorphism from } V \text{ to } V. \quad (n = 0 + \text{range } \implies \text{range} = n)$$

3. What is the ``meaning'' of this isomorphism?
The isomorphism from $\mathbb{K}^n \to \mathbb{K}^n$ with $\varphi_{B'}(\varphi_B^{-1}(x))$ is the change of basis operator.
The inverse $\psi_{B'B}^{-1}$ is $\psi_{B'B}^{-1} = \varphi_B^{-1}(\varphi_B(v))$.

Is there any use of this?
