\documentclass{article}\usepackage{amsmath, amssymb}\begin{document}\setcounter{section}{2}\subsubsection*{3)} Suppose that $T \in \mathcal{L}(F^n, F^m)$.\newline Show that there exists scalars $A_{j,k} \in \mathbb{F}$ for $j=1,\dots,m$ and $k=1,\dots,n$ such that:\newline $$T(x_1,\dots,x_n) = \left( A_{1,1}x_1 + \dots + A_{1,n}x_n,\,\dots,\, A_{m,1}x_1 + \dots + A_{m,n}x_n \right)$$\newline for every $(x_1,\dots,x_n) \in F^n$.\newline \textbf{Solution}\newline Let $B_n = \{e_1,\dots,e_n\}$ be a basis for $F^n$, and $B_m = \{e'_1,\dots,e'_m\}$ be a basis for $F^m$.\newline Since $T$ is linear, we have:\newline $$T(x_1,\dots,x_n) = T\left( e_1 x_1 + \dots + e_n x_n \right) = T(e_1)x_1 + \dots + T(e_n)x_n$$\newline Since $T(e_k)$ are elements of $F^m$, therefore $A_{j,k}$ exist such that:\newline \begin{align*} T(e_1) &= A_{1,1}e'_1 + \dots + A_{m,1}e'_m \\ \vdots \\ T(e_n) &= A_{1,n}e'_1 + \dots + A_{m,n}e'_m \end{align*}\newline and so:\newline $$T(x_1,\dots,x_n) = \left( A_{1,1}e'_1 + \dots + A_{m,1}e'_m \right)x_1 + \dots + \left( A_{1,n}e'_1 + \dots + A_{m,n}e'_m \right)x_n$$\newline Rearranging by the basis vectors $e'_j$:\newline $$T(x_1,\dots,x_n) = \left( A_{1,1}x_1 + A_{1,2}x_2 + \dots + A_{1,n}x_n \right)e'_1 + \dots + \left( A_{m,1}x_1 + A_{m,2}x_2 + \dots + A_{m,n}x_n \right)e'_m$$\newline If we let $e'_j$ be the standard basis vectors in $F^m$, i.e., $e'_1 = \begin{pmatrix} 1 \\ 0 \\ \vdots \\ 0 \end{pmatrix}$, and so on, then the components of the resulting vector are exactly the terms in the parentheses:\newline $$T(x_1,\dots,x_n) = A_{1,1}x_1 + A_{1,2}x_2 + \dots + A_{1,n}x_n,\,\dots,\, A_{m,1}x_1 + A_{m,2}x_2 + \dots + A_{m,n}x_n$$\newline (The original note seems to use a slightly different notation for the resulting vector components in the first display, implying $e'_j$ corresponds to the $j$-th component, which is standard when $e'_j$ is the standard basis.
