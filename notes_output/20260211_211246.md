3) Suppose that $T \in \mathcal{L}(F^n, F^m)$.

Show that there exists scalars $A_{j,k} \in \mathbb{F}$ for $j=1,\dots,m$ and $k=1,\dots,n$ such that:

$$T(x_1,\dots,x_n) = \left( A_{1,1}x_1 + \dots + A_{1,n}x_n,\,\dots,\, A_{m,1}x_1 + \dots + A_{m,n}x_n \right)$$

for every $(x_1,\dots,x_n) \in F^n$.

**Solution**

Let $B_n = \{e_1,\dots,e_n\}$ be a basis for $F^n$, and $B_m = \{e'_1,\dots,e'_m\}$ be a basis for $F^m$. Since $T$ is linear:

$$T(x_1,\dots,x_n) = T\left( e_1 x_1 + \dots + e_n x_n \right) = T(e_1)x_1 + \dots + T(e_n)x_n$$

Since $T(e_k)$ are elements of $F^m$, therefore coefficients $A_{j,k}$ exist such that:

$$T(e_k) = A_{1,k}e'_1 + \dots + A_{m,k}e'_m$$

Specifically:

$$T(e_1) = A_{1,1}e'_1 + \dots + A_{m,1}e'_m$$
$$\vdots$$
$$T(e_n) = A_{1,n}e'_1 + \dots + A_{m,n}e'_m$$

And so:

$$T(x_1,\dots,x_n) = \left( A_{1,1}e'_1 + \dots + A_{m,1}e'_m \right)x_1 + \dots + \left( A_{1,n}e'_1 + \dots + A_{m,n}e'_m \right)x_n$$

Grouping terms by the basis vectors $e'_j$:

$$T(x_1,\dots,x_n) = \left( A_{1,1}x_1 + A_{1,2}x_2 + \dots + A_{1,n}x_n \right)e'_1 + \dots + \left( A_{m,1}x_1 + A_{m,2}x_2 + \dots + A_{m,n}x_n \right)e'_m$$

If we take $B_m$ to be the standard basis for $F^m$, where $e'_j$ is the vector with $1$ in the $j$-th position and $0$ elsewhere (e.g., $e'_1 = \begin{pmatrix} 1 \\ 0 \\ \vdots \\ 0 \end{pmatrix}$), then the components of the resulting vector $T(x_1, \dots, x_n)$ are exactly the coefficients of the $e'_j$ terms:

$$T(x_1,\dots,x_n) = \begin{pmatrix} A_{1,1}x_1 + A_{1,2}x_2 + \dots + A_{1,n}x_n \\ A_{2,1}x_1 + A_{2,2}x_2 + \dots + A_{2,n}x_n \\ \vdots \\ A_{m,1}x_1 + A_{m,2}x_2 + \dots + A_{m,n}x_n \end{pmatrix}$$

This matches the form implied by the first required equation, where the $j$-th component is $\sum_{k=1}^n A_{j,k} x_k$.
